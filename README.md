🍽️ Restaurants UI Collection

A curated collection of modern restaurant website templates maintained by Xenotrix Technologies. This repository includes multiple ready-to-use UI designs for restaurants, cafes, bakeries, and food businesses.

🚀 Features
🍕 Multiple restaurant UI designs
☕ Cafe, bakery, hotel & food service templates
📱 Fully responsive layouts
🎨 Modern and clean UI/UX
⚡ Ready-to-use static websites
💻 Built with HTML, CSS, JS (Bootstrap-based)
